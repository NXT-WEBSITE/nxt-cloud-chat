# Contributing to NXT Cloud Chat

Thanks for your interest in contributing to NXT Cloud Chat.

## Ways to Contribute

- Report bugs
- Suggest features
- Improve documentation
- Share workflow ideas
- Help test new releases
- Participate in GitHub Discussions

## Before Submitting Issues

Please:
- Check existing issues first
- Include clear reproduction steps
- Mention WordPress, PHP, and plugin versions
- Include screenshots/logs when possible

## Feature Requests

Feature requests are welcome.

Good requests usually include:
- use case
- expected behavior
- workflow example
- business/problem being solved

## Pull Requests

Small fixes and documentation improvements are welcome.

Please:
- keep pull requests focused
- follow WordPress coding standards where practical
- avoid unrelated formatting-only changes

## Security Issues

Please do not publicly disclose security vulnerabilities.

Report security-related issues privately via:
support@nxtwebsite.com

See SECURITY.md for more details.

## Discussions

Use GitHub Discussions for:
- questions
- ideas
- workflows
- integrations
- automation concepts
- community sharing

## Pro Features

Discussion about Pro features is welcome, including:
- workflows
- automation ideas
- WooCommerce integrations
- expected behavior
- feature requests

The Pro add-on source code itself is private and commercially licensed.
